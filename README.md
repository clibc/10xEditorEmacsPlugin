# 10xEditorEmacsPlugin

#### Features
- Mark emulation
- Block jump

#### Commands
- `EmacsSetCursor`
- `EmacsCopy`
- `EmacsCut`
- `EmacsDelete`
- `EmacsCursorJumpBack`
- `EmacsScrollUp`
- `EmacsScrollDown`
- `EmacsJumpBlockUp`
- `EmacsJumpBlockDown`

Scroll amount changed via `ScrollLineCount` variable in `Emacs.py`.

#### Installation
Copy `Emacs.py` to `%appdata%/10x/PythonScripts` .

Set keyboard shortcuts in 10xEditor.
